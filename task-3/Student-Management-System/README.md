# 🎓 Student Management Dashboard

A responsive and interactive Student Management Dashboard built using **HTML, CSS, and JavaScript**.
This project allows users to manage student records efficiently with features such as adding, editing, deleting, searching, attendance tracking, and dark mode. Student data is stored using the browser's Local Storage, ensuring that records remain available even after refreshing the page.

## 🚀 Features

- Add new students
- Edit existing student details
- Delete student records
- Search students by name
- Toggle attendance (Present/Absent)
- Dashboard statistics
  - Total Students
  - Present Students
  - Absent Students
- Dark/Light Mode
- Local Storage support
- Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
