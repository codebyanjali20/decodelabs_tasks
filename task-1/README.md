# VisionByte – Static Webpage Design

> A modern static landing page developed as **Project 1** during my **Frontend Development Internship**.

##  Overview

VisionByte is a static web page created using **HTML5** and **CSS3** as part of my Frontend Development Internship. The project focuses on building a clean, visually appealing, and well-structured user interface while applying core frontend development concepts and best coding practices.

---

##  Project Objectives

- Develop a responsive static webpage using HTML and CSS.
- Apply semantic HTML for better structure and accessibility.
- Design an attractive and user-friendly interface.
- Strengthen understanding of CSS layouts, styling, and positioning.
- Build a solid foundation in frontend web development.

---

## Key Features

- Modern landing page design
- Responsive navigation bar
- Hero section with call-to-action button
- Login form interface
- Clean typography and color palette
- Well-organized HTML structure
- Responsive layout for different screen sizes
- Hover effects for interactive elements

- HTML5
- CSS3
- Visual Studio Code
